# minimum_spanning_tree  

###### 最小全域木問題を解くコード  

    $ python minimum_spanning_tree.py --node_fn <total node>

`--node_fn <total node>`の部分でノードの数を指定する  
指定しなければ5つのノードの場合で計算する  


実行例  

    $ python minimum_spanning_tree.py --node_fn 5
