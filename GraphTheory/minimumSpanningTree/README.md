# minimumSpanningTree  

最小全域木問題を解くコード  

### Usage
    $ python minimumSpanningTree.py --node <total node>

`--node <total node>`の部分でノードの数を指定する  
指定しなければノードが5つの場合で計算する  


### Example

    $ python minimumSpanningTree.py --node 5
