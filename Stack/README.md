# Stack
スタックを利用した数式の計算

#### スタックマシンの計算方法
**演算子が登場したらスタックから2つのデータを取り出し、計算したものをスタックに積む**

`1 2 +`
> 3 (=1+2)

1 2 -
> -1 (=1-2)

2 4 /
> 0.5 (=2/4)


---

**スタックに積む順番を変更することで`()`の計算も区別可能**  

(1+2)*3
> 1 2 + 3 *

1+(2*3)
> 1 2 3 * +



#### Example
`2+(3/4-1)*8`を計算する
> 2 3 4 / 1 - 8 * +
```
>>2
now stack: 2
>>3
now stack: 2, 3
>>4
now stack: 2, 3, 4
>>/
now stack: 2, 0.75
>>1
now stack: 2, 0.75, 1
>>-
now stack: 2, -0.25
>>8
now stack: 2, -0.25, 8
>>*
now stack: 2, -2
>>+
now stack: 0
>>=
result = 0
```
